aplicação utilizando os conceitos de POO que manipule uma grande quantidade de dados estatísticos

Dados escolhidos:
- like,views,publicação,comentários

informações extraídas dos dados:
- média
- moda
- mediana
- variância
- desvio padrão

entrada: txt ou web
saída: html

funcionalidade:
- persistência em outra base (banco de dados, xml, json)

- se o html não foi mudado, o programa não gera novamente
